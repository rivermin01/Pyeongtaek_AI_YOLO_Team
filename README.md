# 깃허브 사용 방법

## 초기 설정
1. 계정 생성
2. 깃 설치
3. 바탕화면에 폴더 생성
4. CMD로 폴더 디렉토리 변경
5. git clone https://github.com/rivermin01/Pyeongtaek_AI_YOLO_Team.git
6. CMD로 방금 생긴 폴더 디렉토리 변경
7. git checkout Develop


## 업데이트 방법
1. 파일 수정
2. git add .
3. git commit -m "바꾼 점 요약"
4. git push --set-upstream origin Develop


## 누군가 업데이트 후
1. git pull
